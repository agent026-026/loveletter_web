产品说明

「爱的信件」（Love Letter）是一款在 iPhone 上使用的书信式体验应用：你可以为一位「笔下人物」设定称呼、关系与性格，用信件与对方交流；当你寄出自己的信后，应用会通过 OpenRouter 调用你所选的 AI 模型，生成一封助理回信。回信在随机等待时间结束后可读，以模拟「信件在途中」的节奏。应用侧重陪伴与共情表达，适合希望以文字整理情绪、延续记忆或练习温柔对话的用户。

使用本应用需要你在设置中填写有效的 OpenRouter API Key，并保证网络可访问 OpenRouter。开发者不运营用于中转或长期存储你通信内容的自有服务器；信件与人物设定默认保存在本设备。

功能概要

  • 为通信对象设定名称、关系摘要、性格与称呼方式，并可填写可选的「回信书写要求」。
  • 以书信形式撰写与阅读往来；AI 回信生成后进入锁定状态，在约 5 分钟至 8 小时之间的随机整分钟间隔后解锁可读（每封信不同）。
  • 使用用户自备的 OpenRouter API Key；可在设置中配置模型 ID（默认示例为 openai/gpt-4o-mini）。
  • API Key 保存在系统钥匙串；信件与对象资料由 SwiftData 存于本机。可选开启本地通知，在来信可读时提醒；通知文案为模糊提示，不包含信件正文。
  • 支持浅色 / 深色 / 跟随系统外观；可在设置中删除本机全部通信数据（不自动清除钥匙串中的 API Key）。

隐私与数据

以下为「爱的信件」在处理你的内容与调用 AI 时的数据说明。关于 OpenRouter 及各模型提供方对 API 数据的处理，须以其 隐私政策、服务条款 及各服务商公开文档为准。

我们处理哪些数据

  • 你在设备上创建或编辑的信件正文、发送状态、解锁时间以及与线程关联的对象资料（如称呼、关系与性格描述、可选回信要求等）默认保存在本机，用于应用内的展示与管理。
  • 你在设置中填写的 OpenRouter API Key 保存在系统钥匙串；所选模型 ID、外观偏好等保存在本机用户偏好中，用于组装网络请求与界面。
  • 本应用不会在设备上收集与你身份绑定的生物特征数据；AI 回信功能处理的是你主动输入的文本（信件与上述对象设定中的文字），而非用于人脸识别的图像模板上传（本应用核心流程不涉及此类数据）。
  • 仅在你主动寄出用户信并成功触发「生成回信」时，应用会将当次请求所需的文本内容（通常包括：为模型编写的系统提示中的对象设定摘要、往来信件中已发送且仍保留在应用内的对话文本，以及为维持对话结构而使用的占位说明）通过加密连接（HTTPS）直接发送至 OpenRouter；OpenRouter 会根据其路由规则将请求转发至你所选模型对应的提供方。
  • 若你授权本地通知，系统仅在 scheduled 时间触发提醒；通知内容不包含信件正文，开发者不通过通知渠道收集额外通信内容。

我们如何使用这些数据

  • 仅用于调用第三方大语言模型，将你的来信与上下文转化为助理回信文本，并在应用内按产品设计展示（含锁定与解锁流程）。
  • 不会将你的内容用于开发者侧的广告定向、用户画像构建，或与本应用「书信与回信」功能无关的目的；开发者不运营用于中转或存储你通信内容的自有后端服务器。

是否与第三方共享、数据路径

  • 会。为实现 AI 回信，在你发起请求时，相关文本会先由你的设备发送至 OpenRouter（openrouter.ai），再由其转发至所选模型提供方。数据路径为：你的 iPhone / iPad 设备 → OpenRouter → 所选 AI 模型服务。
  • 各环节服务商对该等数据的存储、日志与留存政策由其各自条款约束；我们不控制第三方内部日志细则，请务必查阅其官方说明。

保留期限

  • 你的设备上：信件与对象资料保存在应用沙盒内，直至你使用「删除所有通信数据」、删除应用或清除应用数据；API Key 在钥匙串中保留直至你在设置中移除或系统清除钥匙串项。
  • OpenRouter / 模型侧：请求与响应的处理与留存遵循对应产品的现行政策，请以 OpenRouter 及各模型提供方文档为准。

你的选择与撤回

  • 你可随时在设置中移除 API Key或停止使用回信生成；未发起网络请求时，应用不会向 OpenRouter 传输用于生成回信的上述文本。
  • 你可关闭或撤销通知授权；可在设置中删除本机全部通信数据（API Key 与部分界面设置可能保留，详见应用内说明）。
  • 请确保你对输入内容拥有合法权利，勿提交违法、侵权或保密义务不允许外传的信息。

AI 与生成式内容声明

本应用的核心能力依赖生成式人工智能（大语言模型）生成助理回信。使用本功能前，请认真阅读以下声明。

生成内容的性质

  • 应用中展示的助理回信、语气与情节建议等均由 AI 模型自动生成，并非真人撰写，也未经人工事实核查。
  • 输出可能存在错误、不贴切、重复、情绪化过度或不足、或与你的真实经历不符等情况，亦可能反映训练数据中的统计性偏见；请务必自行辨别，将其视为辅助表达工具而非客观事实或承诺。

非专业建议

  • AI 生成的文字不构成且不能替代心理咨询、医疗、法律、危机干预或其他专业服务。
  • 若你正在经历心理危机、家庭暴力风险、自伤或伤人风险等，请立即联系当地紧急服务或专业机构；本应用无法监测或干预现实危险。

你的责任与合理使用

  • 你对输入内容的合法性及基于生成内容所产生的行为负责；请勿将输出用于违法、欺诈、骚扰或侵犯他人（含逝者相关权益与肖像名誉等）的行为。
  • 模型可用性、输出风格与政策以 OpenRouter 及所选模型提供方的当前服务为准；因第三方服务变更、地域限制或网络原因导致的不可用，以服务商规则为准。
