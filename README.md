# 爱的信件（Love Letter）

**用书信，和心里那个人慢慢说话。**

有些关系已经结束，有些话却还想被听见。「爱的信件」是一款以**书信**为主的陪伴式应用：你为自己笔下的人物写下称呼、关系与性格，像真的在等邮差一样，把心情写成信寄出去。回信由你自选的 AI 模型生成，在一段「邮途」时间之后启封——不是即时聊天，而更像**被时间温柔拉开的对话**。

---

## 适合谁

- 想用**长文字**整理情绪、纪念一段关系，而不是滑屏刷回复的人  
- 希望对话有**仪式感**（寄出、等待、拆信），愿意给自己一点留白  
- 重视**数据留在自己手机上**，并愿意自备 OpenRouter API、自主选择模型  

---

## 你会体验到什么

- **人物与口吻**：设定对方怎么称呼你、关系与说话方式，还可写上你希望回信遵循的书写要求。  
- **真正的「信」**：你写信、对方回信；回信生成后会先进入锁定，在约 **5 分钟至 8 小时**之间的随机整分钟后再可读，每封信不同。  
- **温柔提醒**：可选开启本地通知，在来信可读时轻轻提醒；**通知里不会展示信件正文**。  
- **你的钥匙，你的模型**：使用 **OpenRouter**，在应用内填写自己的 API Key 与模型 ID；开发者**不运营**用来存你信件内容的自有服务器。  

---

## 重要说明（请在使用前了解）

- 回信由 **AI 自动生成**，不是真人撰写，也可能不贴切或不符合你的期待；请当作**辅助表达与陪伴**，自行辨别与取舍。  
- **不能替代**心理咨询、医疗、法律或危机干预。若你正处于紧急危险或严重心理危机中，请优先联系当地紧急服务或专业机构。  
- 生成回信时，相关文本会通过加密连接发送至 **OpenRouter** 及你所选模型服务商；完整说明见下方「公开说明页」。
